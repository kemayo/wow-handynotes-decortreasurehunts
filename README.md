# HandyNotes: Decor Treasure Hunts

When you pick up a decor treasure hunt quest in your housing neighborhood, this will show an icon on your map explaining where the riddle is sending you.

If you want to see where all the possible treasures are, you can toggle that on in the options. Otherwise, it's pretty bare-bones.

Note that this **requires** [HandyNotes](https://www.curseforge.com/wow/addons/handynotes), so go install it as well.
